# Eco-Connect
O projeto EcoConnect visa conectar doadores a iniciativas ambientais confiáveis, fortalecendo ações alinhadas ao ODS 15.a, voltadas à conservação da biodiversidade e ao uso sustentável dos ecossistemas. Seu objetivo geral é desenvolver uma plataforma web transparente e funcional que facilite o cadastro, a gestão e o apoio a projetos ambientais. A metodologia adotada envolveu o uso de Node.js com Express para construção da API e a migração planejada do banco MySQL para SQLite3, a fim de simplificar o desenvolvimento, reduzir dependências e melhorar a portabilidade. Foram implementadas rotas REST para autenticação de usuários, registro e listagem de projetos e processamento de doações, integradas a um frontend elaborado em HTML, CSS e JavaScript puro. A aplicação realiza consultas dinâmicas ao banco, atualiza estatísticas em tempo real e apresenta as informações de forma organizada e responsiva. Como resultados, o sistema permite cadastrar usuários e organizações, registrar novos projetos, apresentar dados consolidados de arrecadação e simular contribuições de forma transparente. Conclui-se que a plataforma demonstra a viabilidade do uso de tecnologias acessíveis para promover a mobilização de recursos ambientais, oferecendo uma solução integrada, educativa e alinhada às necessidades de monitoramento e apoio a iniciativas sustentáveis.
